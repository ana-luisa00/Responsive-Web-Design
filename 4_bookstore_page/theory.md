# Div element
The div element is used as a container to group other HTML elements. You will mainly use the div element when you want to group HTML elements that will share a set of CSS styles.

# Class attribute
The class attribute is used to identify one or more elements for styling. Class names do not need to be unique: multiple elements can share the same class.
```
<p class="example">example paragraph</p>
```

# Id attribute
The id attribute adds a unique identifier to an HTML element. Each id should be unique within a page and should only be used once. id values cannot contain spaces and should only contain letters, digits, underscores, and dashes.
```
<p id="para">example paragraph</p>
```

# Button element
The button element is used to create clickable buttons on a webpage. Buttons are interactive elements that users can click to perform actions.
```
<button>Buy Now</button>
```
